# CryptoAlert

Telegram bot for private per-user crypto watchlists with price-threshold and 1h percent-change alerts, morning digests, quiet hours, and aggregated owner metrics.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
